## 🧩 How It Works

1️⃣ User uploads an image
2️⃣ User asks a question
3️⃣ Image + question sent to AI model
4️⃣ AI analyzes and responds
5️⃣ Result displayed instantly

## 🚀 Quick Setup

```bash
git clone https://github.com/yourusername/Enhancing-Visual-Question-Answering-VQA-.git
cd Enhancing-Visual-Question-Answering-VQA-
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 🔑 Add API Key

Create `.env` file:

```
GOOGLE_API_KEY=your_api_key_here
```

### ▶ Run

```bash
python app.py
```
